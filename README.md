# 영어 · 맞춤법 학습 (English & Korean Spelling Quiz)

퀴즈로 반복하며 공부하는 무료 학습 웹앱입니다. 설치 없이 브라우저에서 바로 사용하세요.

## 학습 내용
- **기초 회화 단어** 약 1,000개 — 일상 대화에 쓰는 기본 단어
- **토익 빈출 단어** 약 1,200개 — 비즈니스·시험 어휘
- **한글 맞춤법** 약 300문제 — 되/돼, 안/않, 사이시옷, 헷갈리는 단어 등

## 특징
- 뜻 맞히기 → 단어 맞히기 → 철자 맞히기 순으로 **자동 출제** (방식 선택 불필요)
- 틀린 문제는 세션 끝에 **다시 출제**되어 복습
- 영어 단어는 **발음 듣기** 지원
- 연속 학습일·누적 정답 등 진행 상황을 브라우저에 자동 저장

## 사용 방법
[GitHub Pages 주소](https://kingofthestock.github.io/english-study/)에 접속하면 됩니다. PC·모바일 모두 지원합니다.
